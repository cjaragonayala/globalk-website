Method to learn anything

You must follow these steps:

  1. Write the set of high level learning objectives.

  2. Review good information sources.

  3. Specify the low-level (operational) learning objectives (Bloom’s Taxonomy)

  4. Elaborate a good learning material.

  5. Elaborate a good learning assessment.
