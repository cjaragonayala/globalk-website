Install vim plugins

After install vim-plug:

  1. Edit '.vimrc' and add the name of source between call end/start.

  2. Restar vim session

  3. Execute ':PlugInstall' in a vim session

  4. You’re done!

Using macros

  1. Type q (to start record)

  2. Type <macro-name>

  3. Do the steps to be recorded

  4. Type q (to end record)

  5. To play a macro: type <n-times-of-plays>@<macro-name> (time of plays is optional)

What is behind a plugin? Where are they?

  1. We need a directory to store plugins.

  2. Every time you open a vim session it is needed to load plugins, so add a line of code in vimrc for that ("packloadall") .
